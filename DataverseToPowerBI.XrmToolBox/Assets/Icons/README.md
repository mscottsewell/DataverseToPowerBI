# Ribbon Icons

This folder contains 20x20 PNG icons for the XrmToolBox toolbar buttons.

## Required Icons

| Filename | Purpose | Suggested Design |
|----------|---------|------------------|
| `FolderIcon.png` | Change Working Folder | Yellow folder |
| `TableIcon.png` | Select Tables | Green/blue grid |
| `BuildIcon.png` | Build Semantic Model | Blue gear |
| `CalendarIcon.png` | Configure Date Table | Calendar with red header |
| `SettingsIcon.png` | Settings | Gray gear |
| `RefreshIcon.png` | Refresh/Sync | Blue circular arrow |
| `ModelIcon.png` | Semantic Model Selector | Blue database cylinder |

## Specifications

- **Size**: 20x20 pixels
- **Format**: PNG with transparency (alpha channel)
- **Color Palette**: Use consistent colors across icons
  - Blue: `#5082C8` (accent)
  - Yellow/Gold: `#FFC864` (folder)
  - Green: `#64B464` (tables)
  - Red: `#C85050` (calendar header)
  - Gray: `#646464` (settings)

Replace the placeholder icons with your custom designs.
